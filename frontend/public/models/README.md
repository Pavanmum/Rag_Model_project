# Brain 3D Model Setup

## How to Download the Brain Model from Sketchfab

1. **Visit the Sketchfab page:**
   https://sketchfab.com/3d-models/brain-3022543bb7a54b88b892ff4907e50929

2. **Download the model:**
   - Click the "Download 3D Model" button on the Sketchfab page
   - You may need to create a free Sketchfab account
   - Select "Autoconverted format (glTF)" as the download format
   - Download the ZIP file

3. **Extract and place the model:**
   - Extract the downloaded ZIP file
   - Find the `.glb` or `.gltf` file inside
   - Rename it to `brain.glb`
   - Place it in this directory: `public/models/brain.glb`

4. **Refresh your application:**
   - The brain model will automatically load
   - If the model doesn't load, check the browser console for errors

## Alternative: Use the Fallback Brain

If you don't want to download the model, the component will automatically create a procedural brain with blood vessels as a fallback.
